# PicSche LP作成プロンプト（Google Stitch用）

## プロジェクト概要

PicSche（ピクスケ）のランディングページを作成してください。PicScheは、学校のお便りや行事予定表を撮影するだけでAIが予定を自動抽出し、iOSカレンダーに登録できるモバイルアプリです。**iOS版とAndroid版の両方に対応**しています。

**ターゲットユーザー**: 小学生〜中学生の子どもを持つ忙しい保護者（特に30〜45歳の母親・父親）

**核心的な価値提案**: 「学校からのお便り、もう手で打ち込まない。」

---

## LP全体の戦略

### マーケティング目標
1. **即座の行動喚起**: App Store / Google Playダウンロードボタンのクリック率最大化
2. **信頼構築**: AIの精度とセキュリティへの不安を払拭
3. **緊急性の創出**: 「今すぐ」ダウンロードする理由を提示
4. **差別化の明確化**: TimeTreeなど競合との違いを暗に示す

### コンバージョンファネル
1. **ヒーローセクション** → 痛点への共感 → 興味喚起
2. **問題提起** → 解決策提示
3. **機能説明** → 具体的な使い方のイメージ
4. **社会的証明** → 信頼獲得
5. **料金** → 無料から試せる安心感
6. **CTA** → ダウンロード

### 9. FAQセクション
**目的**: 購入前の疑問を解消

**見出し**:
```
よくある質問
```

**アコーディオン形式（8問）**:

**Q1: どんな画像に対応していますか？**
A: 学校からのお便り、月間カレンダー、習い事のスケジュール、イベントチラシ、ゴミ出しカレンダー、手書きの予定表など、幅広い画像に対応しています。

**Q2: iOS版とAndroid版で機能に違いはありますか？**
A: 基本機能は同じです。iOS版はiOSの標準カレンダーアプリに、Android版はGoogleカレンダーに直接登録できます。どちらも家族との共有に対応しています。

**Q3: 手書きの文字も読み取れますか？**
A: はい、手書き文字にも対応しています。ただし、極端に崩れた文字や薄い文字は認識できない場合があります。その場合は画像補足欄にヒントを入力すると精度が向上します。

**Q4: 個人情報は安全ですか？**
A: 画像は予定抽出後すぐに削除され、サーバーに保存されません。また、氏名や住所などの個人情報は意図的に抽出しない設計になっています。

**Q5: 無料プランと有料プランの違いは？**
A: 無料プランは月1回まで（広告なし）利用可能です。有料プランは広告なしで、ベーシックは月10回、プレミアムは月20回まで利用できます。すべてのプランで機能は同じです。

**Q6: 精度はどのくらいですか？**
A: 印刷されたお便りであれば95%以上の精度で抽出できます。手書きや複雑なレイアウトの場合は、画像補足欄にコメントを添えることで精度が向上します。

**Q7: 他のカレンダーアプリと何が違うのですか？**
A: TimeTreeなどの汎用カレンダーアプリは全て手入力が必要ですが、PicScheは写真を撮るだけで予定を自動抽出します。また、iOSの標準カレンダーアプリに直接登録するため、専用アプリの立ち上げが不要です。

**Q8: 解約はいつでもできますか？**
A: はい、いつでもApp Store / Google Playの設定から解約できます。解約後も期間終了まで有料機能を利用できます。

**デザイン要件**:
- 質問クリックで回答が展開
- 質問は太字、回答は通常フォント
- 背景: 薄いグレー #F9F9F9
